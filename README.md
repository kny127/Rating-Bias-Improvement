# 구글맵 치킨집 리뷰 데이터를 활용한 요기요 배달 앱 리뷰 평점 편향 개선 모델
[기계학습입문] 과목의 팀 프로젝트 코드입니다.    
담당 역할:
  * 데이터 수집 및 전처리
  * 모델 구현: Random Forest, LightGBM
  * 불균형 데이터 처리: Back Translation, Easy Data Augmentation(EDA) 기법 적용

## 개요
요기요 배달 앱의 과도하게 편향된 평점 분포(5점에 대부분의 리뷰 데이터가 몰려있음)를 개선하기 위한 머신러닝 프로젝트입니다.리뷰 이벤트로 인해 요기요 배달 앱의 리뷰 평점은 다른 플랫폼(네이버 지도, 구글 지도)에 비해 과도하게 우편향된(5점에 몰려있음) 분포를 보입니다. 리뷰 이벤트의 영향으로 부정적 키워드가 리뷰에 적혀있음에도 불구하고 높은 평점이 부여되는 현상, 즉 실제 리뷰 내용과 평점의 불일치가 요기요 리뷰 데이터에서 발견됩니다.본 프로젝트는 가장 편향이 적은 구글 지도의 치킨집 리뷰 데이터로 학습한 모델을 요기요 리뷰 텍스트에 적용하여, 리뷰 내용과 평점 간의 불일치를 해소하고 보다 신뢰성 있는 평점을 예측하고자 합니다.

## 불균형 데이터 처리
### 3단계 데이터 증강 파이프라인
1. Back Translation (평점 1,2: 4개 언어 / 평점 3: 3개 언어 / 평점 4: 2개 언어)
2. Easy Data Augmentation (EDA): 평점 1,2에 문장당 2개 증강 문장 추가
3. augment_text_with_swap: 무작위로 단어들의 위치를 바꿔 원본 데이터에서 변형된 데이터를 생성   
### 결과
* 초기: 27,893개 → 최종: 119,278개 데이터
* 평점별 분포 개선 (1점: 23.12%, 2점: 12.35%, 3점: 17.53%, 4점: 27.22%, 5점: 19.78%)

## 모델
6가지 모델 성능 비교:
  * Machine Learning: Naive Bayes, SVM, Random Forest, LightGBM
  * Deep Learning: CNN, LSTM         
-> Random Forest의 성능이 가장 좋음 (Train Accuracy: 98.77%, Test Accuracy: 92.31%)
## 결과 분석-키워드 분석
실제 값보다 예측 값이 낮은 요기요 리뷰 데이터 : 약 19% 
### 분석 방법
1. Counter 기반: 낮게 예측된 5점 리뷰의 빈도수 상위 30개 키워드 추출
2. TF-IDF 기반: 낮게 예측된 5점 리뷰에서 정확히 예측된 5점 리뷰 대비 2배 이상 가중치를 가진 키워드 추출
### 분석 결과
* 낮게 예측된 5점 리뷰의 부정 키워드: '좀', '짜다', '아쉽다', '눅눅하다'
* 정확히 예측된 5점 리뷰의 긍정 키워드: '맛있다', '좋다', '빠르다', '항상', '감사하다', '최고'
